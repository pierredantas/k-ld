# notes/

Working notes for the K-ESBMC / `kesbmc` manuscript. Not part of the artifact proper;
kept here (rather than under the gitignored `ACM/` paper tree) so they are versioned.

- **`tosem-review.md`** — full peer review of the manuscript against the ACM TOSEM
  bar (ScholarPeer pipeline + reviewer rubric): gating checks, weaknesses W1–W8,
  scores, simulated reviewer reports, and a prioritized revision list.
- **`evaluation-plan.md`** — plan to broaden the evaluation (the main technical ask
  from the review), organized as work packages each closing a named weakness. WP1a
  is implemented and executed in `k-ld/rung8_wp1a/`.
